
# Processo Seletivo - Diel Energia

## Rodando localmente

Clone o projeto

```bash
  $ git clone https://github.com/murilo-alvesmelo/diel-frontend.git
```

Entre no diretório do projeto

```bash
  $ cd diel-frontend
```

Instale as dependências

```bash
  $ npm install
```

Inicie a aplicação

```bash
  $ npm start
```
## Hospedagem

https://diel-frontend.vercel.app/

- react-bootstrap
- react-select
- react-icons
- react-router-dom
